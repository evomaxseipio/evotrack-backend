# GitHub Actions Status Badges

Add these badges to your README.md after pushing to GitHub:

```markdown
[![CI](https://github.com/YOUR_USERNAME/evotrack-backend/actions/workflows/ci.yml/badge.svg)](https://github.com/YOUR_USERNAME/evotrack-backend/actions/workflows/ci.yml)
[![CD](https://github.com/YOUR_USERNAME/evotrack-backend/actions/workflows/cd.yml/badge.svg)](https://github.com/YOUR_USERNAME/evotrack-backend/actions/workflows/cd.yml)
[![codecov](https://codecov.io/gh/YOUR_USERNAME/evotrack-backend/branch/main/graph/badge.svg)](https://codecov.io/gh/YOUR_USERNAME/evotrack-backend)
```

Replace `YOUR_USERNAME` with your actual GitHub username.
